# SolarBoat-DAQ
Real-time marine FDCAN data acquisition on a Raspberry Pi 4B fitted with a CAN FD HAT, using Node-RED to decode CAN FD frames via SocketCAN, visualize live sensor data on Dashboard 2.0, and store time-series data to InfluxDB with edge-to-home server replication.

For members working on their own pi, open the flows.json file, click on Copy raw file button, and then go to Node-Red, click on the menu top right then Import, finally paste the JSON. Or just download the file, import and select the downloaded file in Node-Red.
